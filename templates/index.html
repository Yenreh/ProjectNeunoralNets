Proyecto de Redes Neuronales

Explora diferentes arquitecturas de redes neuronales para clasificación de texto

MLP

Perceptrón Multicapa

Redes neuronales feedforward con capas densas completamente conectadas. Arquitectura simple pero efectiva para clasificación de texto usando representaciones BoW o embeddings.

  • Capas densas
  • Bag of Words
  • Word Embeddings
Ver Modelos MLP

RNN

Redes Recurrentes Simples

Redes neuronales recurrentes básicas que procesan secuencias manteniendo un estado oculto. Capturan el orden de las palabras y dependencias a corto plazo en el texto.

  • Procesamiento secuencial
  • Estado oculto
  • Contexto local
Ver Modelos RNN

LSTM/GRU

RNN con Memoria

Arquitecturas avanzadas con mecanismos de memoria a largo plazo (LSTM) o puertas simplificadas (GRU). Excelentes para capturar dependencias complejas y contexto distante.

  • Memoria a largo plazo
  • Mecanismos de puertas
  • Contexto global
Ver Modelos LSTM/GRU
Sobre el Proyecto

Este proyecto explora diferentes arquitecturas de redes neuronales para la clasificación de reseñas en una escala de 1 a 5 estrellas.

Dataset

200,000 reseñas de productos etiquetadas con estrellas

Frameworks

TensorFlow y PyTorch para entrenamiento de modelos

Evaluación

Métricas de precisión, F1-Score y matriz de confusión
